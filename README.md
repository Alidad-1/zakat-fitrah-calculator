🕌 Fitrah & Zakat Calculator 2026

A simple web application to calculate Fitrah and Zakat amounts for Ramadan 1447.

This tool helps users quickly calculate the amount they need to pay based on commodities for Fitrah and assets for Zakat.

🚀 Features
Fitrah Calculator

Calculates Fitrah based on different commodities:

Wheat

Barley

Dates

Ajwa

Raisins

User can enter number of persons.

Amount automatically updates.

Total Fitrah amount is calculated instantly.

Zakat Calculator

Calculate Zakat based on:

Gold value

Cash

Liabilities

Automatically calculates 2.5% Zakat.

Shows final payable Zakat amount.

🛠 Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)

Concepts used:

DOM Manipulation

Event Listeners

Real-time input calculations

Basic financial logic
